# Apply Discount Calculator 💸

A simple Python program that calculates the final price of an item after applying a percentage discount.

## 🚀 Features
- Validates user input (price and discount)
- Handles invalid inputs gracefully
- Calculates discount accurately
- Interactive command-line interface

## 🧮 How It Works
The program:
1. Takes user input for price and discount
2. Validates the inputs
3. Applies the discount formula:
   Final Price = Price - (Discount% of Price)
